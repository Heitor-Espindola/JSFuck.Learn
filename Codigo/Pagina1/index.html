O BÁSICO:

No JSFuck, programamos usando apenas 6 caracteres: ! [ ] ( ) +. É estritamente proibido utilizar qualquer coisa além disso, nem mesmo o espaço (' ') é permitido.
Lembre-se que se você apertar o botão 'Sobre', abrirá uma página com outras informações uteis, use Ctrl + C e Ctrl + V (copia e cola) à vontade
Se você apertar o botão 'Bibliotaca', abrirá o meu GitHub com tudo que eu já fiz em JSFuck, evite ficar indo lá para copiar os códigos, pelo menos tente

FUNDAMENTOS

false = ![]

true = !![]

'true' = !![]+[]

undefined = [][[]]

NaN = +[![]]

0 = +[]

1 = +!+[]

2 = !![]+!![]

Considerando que toda string é um Array, podemos fazer:

'f' = ('false')[0] -> 'f' = (![]+[])[+[]]

'a' = ('false')[1] -> 'a' = (![]+[])[+!+[]]

'l' = ('false')[2] -> 'l' = (![]+[])[!![]+!![]]

's' = ('false')[3] -> 's' = (![]+[])[!![]+!![]+!![]]

'e' = ('false')[4] -> 'e' = (![]+[])[!![]+!![]+!![]+!![]]

E então, podemos concatenar para fazer:

'faltar' = (![]+[])[+[]]+(![]+[])[+!+[]]+(![]+[])[!![]+!![]]+(!![]+[])[+[]]+(![]+[])[+!+[]]+(!![]+[])[+!+[]]
