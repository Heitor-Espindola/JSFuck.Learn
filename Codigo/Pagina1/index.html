O BÁSICO:

No JSFuck, programamos usando apenas 6 caracteres: ! [ ] ( ) + . É estritamente proibido utilizar qualquer coisa além disso, nem mesmo o espaço (' ') é permitido.
Lembre-se que se você apertar o botão 'help' ali em baixo, abrirá uma biblioteca de tudo que você já fez em JSFuck, use Ctrl + C e Ctrl + V (copia e cola) à vontade

FUNDAMENTOS

false = ![]

true = !![]

'true' = !![]+[]

undefined = [][[]]

NaN = +[![]]

0 = +[]

1 = +!+[]

2 = !![]+!![]


Considerando que toda string é um Array, podemos fazer:

'f' = ('false')[0] -> 'f' = (![]+[])[+[]]

'a' = ('false')[1] -> 'a' = (![]+[])[+!+[]]

'l' = ('false')[2] -> 'a' = (![]+[])[!![]+!![]]

's' = ('false')[3] -> 'a' = (![]+[])[!![]+!![]+!![]]

'e' = ('false')[4] -> 'a' = (![]+[])[!![]+!![]+!![]+!![]]


E então, podemos concatenar para fazer:

'faltar' = ('false')[0]+('false')[1]+('false')[2]+('true')[0]+('false')[1]+('true')[1] -> 'faltar' = (![]+[])[+[]]+(![]+[])[+!+[]]+(![]+[])[!![]+!![]]+(!![]+[])[+[]]+(![]+[])[+!+[]]+(!![]+[])[+!+[]]
